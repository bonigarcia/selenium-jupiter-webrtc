# Selenium-Jupiter WebRTC [![][Logo]][GitHub Repository]

This project contains several tests aimed to assess videoconferencing applications based on WebRTC with [Selenium-Jupiter], namely:

* [Janus]
* [Jitsi]
* [OpenVidu]
